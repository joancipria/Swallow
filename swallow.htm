Acerca de Swallow

Swallow es un editor de mapas HTML libre construido en tecnologías web Javascript, CSS y HTML y su uso esta enfocado a ser una herramienta de desarrollo web permitiendo dibujar fácilmente sobre una imagen un mapa HTML para su posterior uso en la web.
